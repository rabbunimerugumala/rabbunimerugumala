# 📜 Patents & Publications

## 🏆 Published Engineering Patents

### 1️⃣ Automated Monitoring, Management and Alert System for Real-Time and Backup Power in Apartments or in Communities
### 📖 Description
An embedded IoT-based system for automated real-time monitoring of apartment power supply. Detects power availability using voltage and current sensors, triggers intelligent buzzer and LED alerts to notify residents about generator start/stop requirements, and displays live readings on an LCD — reducing fuel wastage and enabling efficient backup power management in apartments and communities. Filed at the Indian Patent Office. Application No. `202541053612`. Published U/S 11A on **13 June 2025**.

### 2️⃣ Smart LPG Monitoring System for Real-Time Gas Level and Safety Alerting Using Weight-Based Detection
### 📖 Description
A smart LPG safety system using weight-based detection to continuously monitor real-time gas cylinder levels. Calculates remaining gas quantity from weight readings and triggers proactive safety alerts before dangerous depletion or leakage conditions arise — improving household and community gas safety. Filed at the Indian Patent Office. Application No. `202541113812`. Published U/S 11A on **05 December 2025**.
